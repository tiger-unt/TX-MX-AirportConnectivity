This folder contains some code for extracting data from SQLite DB using Python.

The code uses pandas, but there are other ways using the SQLite library and other tools. I have found the Pandas methods to work from most situations.

The LoadBTSData.py file creates a db from a set of zip files downloaded from the BTS website

The SQLite example .....py file contains a few easy functions for pulling data out of the database using SQL and Pandas
